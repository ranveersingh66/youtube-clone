# 🎬 YouTube Clone (HTML + CSS)

A responsive YouTube homepage clone built using pure HTML, CSS.

🔗 **Live Demo:**  
https://ranveersingh66.github.io/youtube-clone/

---

## 📌 Features

- 🎥 Responsive video grid layout (CSS Grid)
- 🔍 Header with search bar UI
- 📂 Sidebar navigation (like YouTube)
- 🖼️ Video thumbnails with rounded corners


---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)
  
---

## 📁 Project Structure
YOUTUBEPROJECT/
├── index.html
